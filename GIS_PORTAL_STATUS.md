# GIS Portal - Master Plan Status

**Location in Roadmap:** Phase 5 - Project Intelligence & Memory System

**Current Phase:** Phase 1 (Foundation & Stabilization)

---

## What is the GIS Portal?

**Vision:** Integration with geographic information systems to overlay project data with:
- Parcel boundaries
- Zoning overlays
- Utility corridors
- Wetlands/environmental constraints
- Flood zones
- Agency jurisdictions

**Purpose:** 
- Automatically link projects to their parcels
- Visual map view of all active projects
- Identify regulatory constraints early
- Site analysis automation

## GIS Portal Requirements (From Master Plan)

**Integration Points:**
```markdown
- [ ] GIS data:
  - Link parcels to projects
  - Overlay zoning, utilities, wetlands
```

**Likely Implementation:**
1. Connect to county/city GIS APIs (COJ, Duval County)
2. Parcel lookup by address or project number
3. Map view showing all projects
4. Constraint overlays (zoning, flood, utilities)
5. Auto-populate project site data

**Dependencies:**
- Phase 1 complete ✅
- Project address/location data standardized
- GIS API access (COJ GIS, SJRWMD)
- Mapping library (Mapbox, Google Maps)
